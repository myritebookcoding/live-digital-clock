# Digital Clock Using HTML, CSS & JavaScript

A modern and responsive digital clock built using **HTML**, **CSS**, and **JavaScript**. This project features animated SVG progress rings, glowing neon effects, and real-time clock updates.

## ✨ Features

* Real-time clock updates every second
* 12-hour AM/PM format
* Animated circular SVG progress indicators
* Glowing neon UI effects
* Fully responsive design
* Perfectly centered text inside circles
* Beginner-friendly code structure

## 📂 Project Structure

```text
digital-clock-project/
│── index.html
│── style.css
└── script.js
```

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* SVG Graphics

## 📸 Preview

The application displays:

* Hours, minutes, and seconds in separate circular rings
* Animated progress indicators
* AM/PM display
* Responsive layout for desktop, tablet, and mobile devices

## 🛠 Installation

1. Clone this repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd digital-clock-project
```

3. Open `index.html` in your web browser.

No additional dependencies or installation steps are required.

## 🎯 Learning Outcomes

By building this project, you will learn:

* JavaScript DOM manipulation
* Working with `Date()` objects
* SVG animations using `stroke-dasharray` and `stroke-dashoffset`
* CSS Flexbox layouts
* Responsive web design
* CSS custom properties and glowing effects
